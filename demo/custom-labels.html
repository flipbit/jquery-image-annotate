Custom Labels

Override button labels for internationalization or icon-only mode.

Default (English)

No labels option — uses built-in defaults.

German Labels

Full label override: labels: { addNote: "Notiz hinzufügen", save: "Speichern", delete: "Löschen", cancel: "Abbrechen", placeholder: "Notiz eingeben..." }

Icon-Only

All labels set to empty strings — buttons render with icons only.
