jQuery Image Annotate

Create Flickr-like comment annotations on images. Draw rectangular regions, add text notes, save and load via AJAX or static data.

  • jQuery Basics Initialize via $.fn.annotateImage() with static notes. Editable vs read-only modes.
  • Vanilla JS Core API without jQuery. Uses annotate() factory function and ES module import.
  • React React 18 component with event callbacks and imperative ref methods.
  • Vue Vue 3 component with event handling and template ref methods.
  • AJAX Load, save, and delete annotations via fetch. Custom error handling.
  • Multiple Instances Several annotated images on one page with different configurations.
  • Programmatic API Control the plugin via JavaScript: add, clear, load, destroy, reinitialize, export.
  • Custom Labels Override button labels for internationalization or icon-only mode.
